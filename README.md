# ポケモン選出支援ツール

ポケモン対戦の選出を、タイプ相性と内蔵データから支援するWebアプリです。

## 使い方

1. 自身のパーティ6体を入力します。
2. 相手のパーティを入力します。
3. 相手の名前が分からない場合は、タイプだけ入力しても判定できます。
4. 「選出する」を押すと、スコアが高い3体と対面させる相手が表示されます。

## 判定に使う情報

- ポケモン名
- タイプ
- 種族値
- 参考技タイプ
- タイプ相性表

## 判定から外した情報

以下は更新頻度が高く、権利面や手入力コストも大きいため、アプリ内の判定・表示から外しています。

- ポケモンチャンピオンズの内定ポケモン一覧
- ポケモンチャンピオンズの採用率
- 性格採用率
- 持ち物採用率
- 技採用率
- 能力ポイント採用率

## データベース

データは `data/pokemon.json` で管理します。
基本DBの元データには `veekun/pokedex` のCSVを利用できます。

主な項目:

- `id`: 管理用ID
- `name`: ポケモン名
- `types`: タイプ
- `baseStats`: 種族値
- `referenceMoveTypes`: 判定に使う参考技タイプ
- `updatedAt`: 更新日

変換方法は `data/README.md` を確認してください。

## 注意

このアプリは選出判断の補助ツールです。技の威力、命中、特性、持ち物、テラスタル、素早さ関係、実際の型読みは最終判断で補ってください。
