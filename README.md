# ポケモン相性判定と選出アドバイス

GitHub Pages などの静的ホスティングにそのまま公開できるWeb版です。

## ファイル

- `index.html`
- `styles.css`
- `app.js`
- `pokemon_selection_app_icon.png`

## 公開方法

1. GitHubで新しいリポジトリを作成します。
2. 上記ファイルをリポジトリ直下にアップロードします。
3. リポジトリの `Settings` から `Pages` を開きます。
4. `Deploy from a branch` を選び、`main` ブランチの `/root` を指定します。
5. 数分待つと公開URLが表示されます。

登録した自身のパーティは、利用者ごとのブラウザ内 `localStorage` に保存されます。
