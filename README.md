# ポケモン相性判定と選出アドバイス

ポケモン対戦の選出を、タイプ相性から支援するWebアプリです。

## 使い方

1. `自身のパーティ（6体）` に、自分のポケモン名とタイプを入力します。
2. `相手のパーティ（6体）` に、相手のポケモン名とタイプを入力します。
3. 相手のポケモン名が分からない場合は、名前を空欄にしてタイプだけ入力しても判定できます。
4. `選出する` を押すと、選出すべき自分の3体が表示されます。

## 入力補助

内蔵データにあるポケモン名を入力すると、タイプと参考技タイプが自動で反映されます。

タイプを選ぶと、そのタイプに一致するポケモン候補が表示されます。候補名を押すと、そのポケモンを入力できます。

## 判定内容

- 自分の各ポケモンについて、相手6体への攻撃倍率を合計します。
- 相手6体から受ける被ダメージ倍率を合計します。
- ポケモン徹底攻略の「覚えている技」を参考にした内蔵の参考技タイプを考慮します。
- 参考技タイプが未設定の場合は、そのポケモン自身のタイプ一致技を持つものとして判定します。
- 攻撃倍率と被ダメージ倍率から総合スコアを計算します。
- 総合スコアが高い3体を選出します。
- 相手の各ポケモンに対して、どの自分のポケモンを対面させるべきか表示します。
- 対応データがあるポケモンは、性格、特性、持ち物、覚えている技、能力ポイントの採用率をもとに注意点を表示します。

## パーティ登録

`登録名` を入力して `登録` を押すと、自身のパーティを保存できます。

保存したパーティは `読込` で呼び出せます。不要になったパーティは `削除` で消せます。

登録データは、使用しているブラウザ内に保存されます。

## 育成論の参考リンク

選出結果の下に、ポケモン徹底攻略の育成論トップ、詳細検索、選出されたポケモンの育成論検索リンクが表示されます。

選出後に、技構成、持ち物、性格、努力値などを確認したい場合に使えます。

## 採用率に基づく注意点

内蔵データに一致するポケモン名がある場合、結果欄に対戦時の注意点が表示されます。

相手パーティに一致するポケモン名がある場合は、相手ポケモンの注意点を優先して表示します。

相手側に一致データがない場合は、選出された自分のポケモンについて注意点を表示します。

## 注意

このアプリはタイプ相性と参考技タイプをもとに判定します。

技の威力、命中、特性、持ち物、テラスタル、素早さ関係は判定に含まれていません。

採用率データは参考情報です。最新データや詳細は、表示されるポケモン徹底攻略への参考リンクから確認してください。
